# Start Each LLM Session With This

## For Amplifier (Port 3000s)
```
I need you to build a portfolio builder web application for creative professionals.

In the plans/ folder, you'll find all the specifications:
- portfolio-vision-ux.md - User experience requirements
- portfolio-tech-strategy.md - Technical suggestions
- portfolio-implementation-plan.md - Phased approach
- user-success-scenarios.md - Real user stories with test credentials
- mockups/ - Interface concepts

In the test-assets/ folder, you'll find realistic test data:
- User profiles for Sarah Chen, Marcus Williams, Emma Rodriguez, and a basic test account
- Sample content, project descriptions, and professional bio templates
- Detailed image generation specs if you want to create realistic portfolio images
- Generic testing scenarios beyond the user personas

The minimum requirement is completing Phase 1 from the implementation plan, but please go as far as you can - completing all 5 phases would be ideal.

For development, please use ports in the 3000 range (3000, 3001, etc.) to avoid conflicts.

Review the documentation and build the best solution you can. The test credentials are in plans/user-success-scenarios.md if you want to pre-seed realistic user accounts.

Let me know if you have any questions before starting.
```
